| 项目 | 说明 |

## 游戏介绍

扮演邦多利世界观的新手工作人员，通过多样的角色组合，在各种各样的致敬与玩梗中用各种各样（怎么癫怎么来）的技能和道具将其他新手工作人员击败，带领你的少女乐队成员成为邦邦武斗传的最终赢家！

## 一、卡牌种类

### 1.1 角色牌

每名玩家在一盘对战中扮演的角色。一张角色牌包含以下要素：

| 要素 | 说明 |
|------|------|
| **血量** | 1~5，角色可以承受的伤害范围 |
| **基础攻击力** | 1~5，使用行动牌时角色能造成的伤害 |
| **所属阵营** | 部分技能和卡牌会涉及阵营（1.0版本首发角色有10个阵营） |
| **技能** | 能为角色提供加值或有助于获胜的效果 |

#### 血量状态

| 状态 | 表示方式 | 说明 |
|------|----------|------|
| **正常状态** | 角色牌正常放置 | 可正常行动 |
| **濒死状态** | 角色牌180°倒置 | 受到任意伤害将直接退场 |
| **退场状态** | 角色移出部署槽进入休息室 | 不在场 |

#### 阵营技

阵营技能够为场上的同阵营角色提供效果加成，触发条件较为苛刻：**己方场上部署的五名角色必须是同阵营**。

> **注意**：当出现两个玩家凑齐一个相同阵营时，阵营技只能给**率先**凑齐该阵营的玩家使用。
>
> 目前1.0版本只有10个阵营技且每个阵营的角色都只有一个本体，不会出现两个玩家同时凑齐一个阵营的情况（除非买两套牌来玩）。

#### 技能类型

| 类型 | 区分方式 |
|------|----------|
| **消耗行动型** | 技能介绍前缀有"消耗行动点" |
| **非消耗行动型** | 技能介绍前缀无"消耗行动点" |

---

### 1.2 行动牌

行动牌主要有4种：**攻击**、**补充**、**格挡**、**回复**。

#### 攻击

| 类型 | 说明 |
|------|------|
| **普通攻击**（肘） | 伤害取决于行动角色的攻击力 |
| **破甲攻击**（破） | 无视对方的格挡和护甲抵消 |
| **肘+20** | 伤害为角色攻击力+20（每花色5张，共20张） |
| **破+4** | 破甲攻击，伤害为角色攻击力+4（每花色1张，共4张） |

> **注**：攻击牌会消耗进行攻击的角色的行动点。

#### 伤害判定机制

对角色进行攻击时：

- **伤害 > 血量** → 角色退场
- **伤害 ≤ 血量** → 根据差值进行概率判定

| 差值（血量 - 伤害） | 濒死概率 |
|----------------------|----------|
| 差值 > 5 | 1/6 |
| 差值 3~5 | 1/3 |
| 差值 < 3 | 1/2 |

> 角色在濒死状态下受到任意伤害将直接退场。

#### 补充

使用该牌可选择以下两种效果之一：

1. **补充一名角色进入手牌**
2. **摸两张牌**

#### 格挡

| 类型 | 效果 |
|------|------|
| **小格挡**（挡） | 帮一名角色抵挡一次伤害 |
| **大格挡**（防） | 帮所有部署的角色抵挡一次伤害 |

两种格挡都是**即时触发，不会保留**。

> **注1**：格挡会消耗行动点，所有角色的行动点共享，没有行动点的角色可以消耗友方角色的行动点来进行格挡。
>
> **注2**：玩家体力值在受到攻击时，也可以使用格挡牌来抵挡伤害（不消耗行动点），但仅限于部署角色为0、对手能够直接攻击体力值的情况下才可使用。

#### 回复

| 类型 | 效果（二选一） |
|------|----------------|
| **小回复** | ① 恢复一名角色的行动点和血量变为正常状态；② 恢复玩家2点体力值 |
| **大回复** | ① 恢复所有部署的角色的行动点和血量变为正常状态；② 恢复玩家5点体力值 |

> 恢复玩家体力值的效果可以在**非己方回合**使用。
>
> 通常情况下，己方回合使用回复牌后不能让该回合已使用行动点的角色继续行动（除非有"该回合没有行动点限制"的效果）。

---

### 1.3 策略牌

策略牌分为两种：**可部署策略牌（道具牌）**和**即时策略牌（即时牌）**。

#### 可部署策略牌（道具牌）

- 能够长期为可部署角色或玩家提供加成或影响对手
- 部署上限取决于已部署角色的数量（一名角色提供一个道具槽，最大部署上限为5）
- 不部署角色将无法装备
- 道具牌部署位置位于角色牌后面（绝大多数道具牌默认只给部署位置的前方角色提供加成）

> **注意**：当部署的道具牌前面对应的角色退场时，如果剩余部署的角色没有空闲的道具槽将会被弃置。

#### 即时策略牌（即时牌）

- 卡牌效果即时触发
- 只要卡牌上的条件允许就能在任何时候使用
- 大部分只能在己方回合使用，极少数能在非己方回合使用

#### V1.1新增策略牌

| 卡牌 | 类型 | 效果 |
|------|------|------|
| **麻里奈的礼物箱** | 即时 | 摸两张牌，若两张牌花色相同则再摸一张（每花色3张，共12张） |

---

### 1.4 辅助牌

辅助牌的作用主要用于区分一些角色技能所产生的独占单位或独有道具等（以及在玩家凑齐阵容时获取对应阵营技）。这类牌**不会添加进手牌堆中**，仅方便玩家记录游戏状态。

## 二、游戏机制

### 2.1 角色行动点

- 每名角色每回合有**1点行动点**
- 行动点被消耗后，通过角色牌顺时针90°横置来表示
- 每次进入己方回合时刷新角色行动点
- **濒死状态的角色默认不提供任何行动点**，即便该角色的行动点尚未使用

### 2.2 角色血量

- 血量状态分为三种（正常/濒死/退场）
- 濒死状态通过角色牌倒置来表示
- 每次进入己方回合时会将濒死状态的角色刷新为正常状态

### 2.3 玩家体力值

- 每名玩家的初始体力值为 **10**
- 玩家可以在己方回合通过消耗 **3点体力值** 来补充1名角色进入手牌
- 每回合限制**两次**抽取

### 2.4 角色牌上限

- 玩家可部署在场地的角色最多 **5名**
- 未部署在场地的、手牌里的角色牌上限为 **3张**
- 角色牌**不占用**普通手牌上限
- 场地满时每回合最多替换 **1名** 角色

## 三、对局流程

### 3.1 游戏开始前

1. 所有玩家先从角色池里每人抽 **5张角色牌** 作为首发阵容
2. 进行直接部署
3. 然后每人摸 **5张牌** 作为初始手牌

#### 角色池与休息室

- 玩家获得新角色的主要途径是抽取角色池
- 当角色池没有可抽选角色后，将无法以常规途径获取新角色
- 部署在场地的角色退场后，将被移入**休息室**（墓地）
- 想要抽取休息室的角色，需要通过策略牌或角色技能来达到目的

---

### 3.2 回合流程

每回合所有玩家的游戏流程如下循环：

#### 1️⃣ 判定阶段

判定分为三种：

| 判定类型 | 说明 |
|----------|------|
| **花色判定** | 卡牌左上角的四种花色 |
| **种类判定** | 行动牌/策略牌、攻击/回复、道具牌/即时牌等 |
| **概率判定** | 投骰子进行，常见概率为1/2、1/3、2/3、1/6~5/6 |

> 一般玩家在自己回合遭到需要判定的事件后，都是先进行判定，再进行摸牌。

**概率判定规则**（以最大数字为成功基准）：

| 概率 | 成功条件 |
|------|----------|
| 1/2 | 骰到 4、5、6 |
| 1/3 | 骰到 5、6 |
| 2/3 | 骰到 3、4、5、6 |
| 1/6 | 骰到 6 |
| 5/6 | 骰到 2、3、4、5、6 |

#### 2️⃣ 摸牌阶段

- 正常情况下摸 **2张牌**
- 刷新部署角色的行动点和血量状态

#### 3️⃣ 行动阶段

玩家在该阶段可以进行以下操作：

1. **部署角色牌到场地**（最多部署5名角色）

   **角色部署机制**：
   - 部署达到上限时进行角色更换，需将要更替的角色收回手牌
   - 更替的角色必须处于正常状态
   - 如要强制更替非正常状态的角色，更替的角色将会**强制退场**进入休息室

2. **消耗角色行动点来使用行动牌或角色技能**（部分策略牌有可能会消耗行动点）

   > 每回合行动点使用限制：玩家在己方回合只能使用 **三名角色** 的行动点。

3. **使用策略牌**（装备道具牌和使用即时牌）

#### 4️⃣ 弃牌阶段

- 正常状态下每名玩家的手牌上限为 **5张**
- 多余的手牌应当弃掉
- **手牌里的角色牌不占用手牌上限**

#### 5️⃣ 回合结束

轮到下一名玩家。

## 四、游戏胜利条件

### 4.1 常规胜利

全场只剩 **1名** 未出局玩家。

### 4.2 全灭胜利

全场全员出局，胜利判给**最后造成这一结果的玩家**。

### 4.3 玩家出局判定

- 玩家体力值正常情况下只有在**部署角色数量为0时**才能直接攻击体力值
- 体力值扣除直接按所受伤害加减来算
- 当玩家体力值归零，且场地上没有可部署的角色时，该玩家出局
- **即便玩家体力值归零了，只要场地上还有部署的角色没退场就能继续游戏**
- 玩家体力值归零后**不能以任何形式恢复体力值**
- 只要确保场地上时刻有一名角色没退场就能继续游戏
- 除此之外玩家仍然能正常按照回合流程进行行动

### 4.4 事件胜利

按照【特殊事件】策略牌的要求完成事件即可获得胜利，该胜利条件难度极高。

> 1.0版本规则暂时没有事件胜利。

## 五、常规负面效果

| 效果 | 说明 |
|------|------|
| **跳过摸牌阶段** | 玩家不能摸牌，直接进入行动阶段 |
| **跳过行动阶段** | 玩家完成摸牌阶段后直接跳过行动阶段进入弃牌阶段 |
| **失去行动能力** | 玩家在自己回合内和自己回合外都将失去操作能力（上述两种跳过阶段加上非己方回合外的不可操作） |
| **破甲/无视护甲** | 无视格挡和护甲抵消两种效果，角色无法通过这两种效果来抵挡伤害 |
| **缴械** | 道具槽里的道具牌失效 |
| **虚弱** | 角色血量减半（四舍五入） |
| **沉默** | 角色无法使用自身技能 |

> **护甲抵消**：通过为角色提供护甲来抵消伤害，每点护甲能抵消1点伤害，只要角色携带有护甲就算作有护甲抵消效果。

## 六、角色技能与卡牌词条

| 词条 | 效果 |
|------|------|
| **守护** | 对方攻击时必须优先攻击己方带有该词条的角色 |
| **无视守护** | 无视带有"守护"词条的角色 |
| **群攻** | 该角色的攻击能对一名玩家的所有部署角色造成伤害 |
| **历战** | 回合开始前角色从前一回合的濒死状态存活，攻击力永久+1，退场后清零 |
| **免伤** | 角色可免疫一次即将到来的伤害 |
| **不可选中** | 策略牌、行动牌和角色技能无法选定拥有该效果的角色来触发 |
| **反击** | 角色被指定为攻击对象时，可对攻击发起者进行一次普通攻击 |
| **地藏** | 将角色牌翻到背面表示该状态。在这一状态下的角色相当于不在场，任何攻击、卡牌效果、加成和减值都不对该角色有用，且该角色也无法使用角色技能和行动点（但该角色装备的道具仍有效）。注意：该状态下会破坏阵营技的触发 |
| **扩散** | 角色技能、角色攻击、卡牌效果等会影响指定角色两旁部署的友方角色 |
